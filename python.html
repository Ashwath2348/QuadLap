Python is a popular, easy-to-learn programming language known for its clear syntax and readability.

2. Key Concepts:

  • Variables: Used to store data. 
    x = 5
name = "Alice"
  • Data Types: int, float, str, bool, list, dict
  • Control Flow:
    • If Statements: 
      if x > 0:
    print("Positive number")
    • For Loops: 
      for i in range(5):
    print(i)
    • While Loops: 
      while x > 0:
    print(x)
    x -= 1
  • Functions: 
    def greet(name):
    return "Hello, " + name
  • Comments: Use # for single-line comments.
  • Indentation: Python uses indentation to define blocks of code.
